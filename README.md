# WaveML-Compress 🌊

A high-performance pipeline architecture combining acoustic physical waveform simulation, dynamic **Huffman Tree Data Structures** written in native **C** for lossless bitstream compression, and a **Python** sliding-window machine learning anomaly engine.

## 🧠 Key DSA & Architectural Concepts Implemented
* **Greedy Binary Trees (Huffman Coding):** Built a custom tree node priority allocator algorithm to convert physical quantized audio streams into variable-length prefix bit representations.
* **Sliding Window Parsing:** Implements a sliding data chunk algorithm in Python to process localized streaming variance metrics.
* **Quantization Matrix Mapping:** Discretizes continuous transcendental trigonometric function equations into integer arrays suitable for token-based frequency structures.

## 🛠️ Project Structure
* `wave_gen.c`: Models multi-frequency physics constructive wave data.
* `huffman_tree.c`: Generates the greedy variable-length encoding binary tree.
* `anomaly_detector.py`: Sliding-window analytical outlier analysis module.
* `pipeline.py`: Governs compilation bindings and evaluates performance metrics.
